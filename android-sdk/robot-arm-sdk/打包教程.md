# SDK 打包教程 (AAR)

本项目使用标准的 Android AAR 格式进行交付。

### 1. 准备环境
确保您已配置好 JDK 17，并在根目录下具有可执行权限的 `gradlew` 文件。

### 2. 执行打包命令
在项目根目录下打开终端，运行：
```bash
./gradlew :android-sdk:robot-arm-sdk:assembleRelease
```

### 3. 获取产物
打包完成后，生成的 AAR 文件存放在以下路径：
`android-sdk/robot-arm-sdk/build/outputs/aar/robot-arm-sdk-release.aar`

---
**提示**：该 AAR 包含了混淆后的代码及资源，可直接交付给客户使用。
